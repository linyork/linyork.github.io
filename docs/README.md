## 🚀 快速開始

### Windows
```cmd
# 回到專案根目錄
cd ..

# 執行啟動腳本
start-dev.bat
```

### Mac/Linux
```bash
# 回到專案根目錄
cd ..

# 執行啟動腳本
./start-dev.sh
```
