## 目的 ##

 - 為了取得完整浮點數完整位數
 
## 紀錄 ##

### 程式 ###

```php
function getTime(){
    list($usec, $sec) = explode(" ", \microtime());
    return $sec+$usec;
}
function calculate($start, $end){
    echo "執行秒數: " . sprintf("%.10f", $end-$start);
}
$start = getTime();

$end = getTime();
calculate($start, $end);
```
### 結果 ###

```
執行秒數: 0.0000003451373
```
